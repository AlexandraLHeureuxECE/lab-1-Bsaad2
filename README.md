# Lab 1 – Tic-Tac-Toe

## Live Demo (GitHub Pages)
https://alexandralheureuxece.github.io/lab-1-Bsaad2/

## How to Run
This project is deployed as a static website using GitHub Pages.
To run locally (optional), you can use a simple static server:
- `python -m http.server 5173`
- then open `http://localhost:5173`

## Features
- 3×3 Tic-Tac-Toe grid
- Two-player gameplay on the same device (X then O)
- Win detection (rows/columns/diagonals)
- Draw detection
- Status text (turn / winner / draw)
- Restart button (no page refresh)
